Package harp-map-theme

@here/harp-map-theme

Overview

This module contains the themes used by harp.gl. Inside the resources folder you will find

  • json files containing the styles definition
  • png file containing the atlas of icons to be used together with the style

Inspiration for the Style

The style is called berlin, inspired by the vibrant city of Berlin, where most of the development of harp.gl is done.

The choice of colors from the city itself, blue from the river Spree, green from the many parks and forests of the city and yellow because of the color of the taxis and the public transport system.

We provide two variants based on the same berlin theme, a dark one and a reduced one.
